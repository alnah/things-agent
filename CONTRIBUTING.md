# Contributing

Thank you for contributing to `things-agent`.

## Contribution policy

- Bugs and small fixes: open a Pull Request.
- New features and architecture changes: start a GitHub Discussion first.
- Built with Codex, Claude, or other AI tools: this is welcome, but you must clearly signal it in the PR/issue.

## Issues and reports

Use the provided GitHub Issue form.

Reports without reproduction steps, impact, and remediation advice are not prioritized.
Given the volume of AI-generated scanner findings, I prioritize vetted reports from contributors who understand and demonstrate the issue.

### Required in reports

- Title
- Severity assessment
- Impact
- Affected component
- Technical reproduction
- Demonstrated impact
- Environment
- Remediation advice

## Pull requests

- Keep changes scoped and atomic.
- Include tests when behavior changes.
- Describe risk and rollback considerations for sensitive changes.
